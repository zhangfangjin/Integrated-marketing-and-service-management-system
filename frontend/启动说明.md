# 前端项目启动说明

## 1. 安装依赖

在 `frontend` 目录下执行：

```bash
npm install
```

## 2. 启动开发服务器

```bash
npm run dev
```

启动后访问：http://localhost:3000

## 3. 确保后端服务运行

前端需要后端 API 支持，确保后端服务运行在 `http://localhost:8080`

## 4. 功能说明

### 登录页面
- 访问 http://localhost:3000/login
- 输入用户名和密码登录

### 注册页面
- 访问 http://localhost:3000/register
- 输入身份证号，系统会自动解析性别、出生日期、年龄
- 选择岗位和所属片区
- 提交后等待管理员审核

### 主界面
- 左侧：系统功能模块树（可展开/折叠）
- 右侧：内容区域，显示当前选中的模块内容

### 模块管理
- 左侧显示模块树
- 点击"增加"按钮可添加子模块
- 右侧显示模块信息表单，可编辑和删除

### 权限配置
- 左侧选择角色
- 中间显示模块树（带复选框）
- 右侧配置选中模块的权限（可浏览、可新增、可修改、可删除等）

### 其他管理页面
- 角色管理：增删改查角色
- 选项管理：管理下拉选项（岗位、片区等）
- 账号管理：审核注册、重置密码、分配角色

## 注意事项

1. 首次使用需要先创建角色和选项数据
2. 注册后需要管理员审核才能登录
3. 确保后端 API 正常运行
